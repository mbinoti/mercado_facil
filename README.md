# Mercado Fácil (Grocery App)

Aplicativo de compras de supermercado feito em Flutter, com fluxo de ponta a ponta para descoberta de produtos, carrinho e checkout.

## Visão geral

O projeto simula a experiência de um app de grocery/delivery com dados locais (seed) usando Hive. Ele inclui telas de onboarding, login, catálogo, detalhes de produto, carrinho e confirmação de pedido.

## Funcionalidades atuais

- Onboarding e login
- Seleção de endereço de entrega
- Home com categorias, ofertas e produtos em destaque
- Busca e listagem por categoria
- Tela de detalhes do produto
- Carrinho com ajuste de quantidade
- Checkout
- Confirmação de pedido
- Persistência local com Hive (dados fake para desenvolvimento)
