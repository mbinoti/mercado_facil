# Docs - app_mercadofacil

Indice rapido dos arquivos de documentacao para reduzir duplicacao e facilitar navegacao.

## Como organizar

- `docs/AGENTS.md`: como executar tarefas no projeto (agentes, fluxo, regras comuns).
- `docs/product/`: contexto de produto, telas/fluxos e imagens de referencia (`screens/`).
- `docs/architecture/`: estrutura tecnica e regras arquiteturais.
- `docs/agents/`: instrucoes detalhadas por agente especializado.

## Leitura recomendada por tipo de tarefa

### Implementacao de feature UI

1. `docs/AGENTS.md`
2. `docs/product/project-context.md`
3. `docs/product/telas-stitch.md`
4. `docs/agents/flutter-specialist.md`

### Mudanca estrutural / arquitetura

1. `docs/AGENTS.md`
2. `docs/architecture/README.md`
3. `docs/architecture/folder_structure.md`
4. `docs/agents/flutter-clean-architecture-agent.md`
